# 変数型
## 整数型
`int`
- 32ビットの符号付き整数。

## 浮動小数点型
`float`
- 単精度浮動小数点数。

## 倍精度浮動小数点型
`double`
- 高精度の浮動小数点数。

## 論理型
`bool`
- 真偽値型。`true`または`false`。

## 文字列型
`string`
- UTF-8エンコードされた文字列。

## 文字型
`char`
- 単一の文字を表す型。例: `'a'`, `'1'`, `'@'`

## 列挙型
`enum`
- 定義済みの定数集合を表す型。
  - 例:
    ```
    enum Color {
        Red,
        Green,
        Blue
    }
    ```

## ユニオン型
`union`
- 複数の型のいずれかを保持できる型。
  - 例: `union { int | string }`

## 空虚型
`void`
- 戻り値がないことを示す型。

## 配列バッファ型
`arraybuffer`
- バイナリデータを効率的に操作するための固定長バッファ。
  - 例:
    ```
    const buffer: arraybuffer = arraybuffer(1024); // 1024バイトのバッファを作成
    buffer[0] = 255; // バッファの先頭に値を設定
    const value: int = buffer[0]; // バッファの先頭から値を取得
    ```
- バッファの長さは作成時に固定され、変更できません。
- バッファの内容は数値型（`int`または`float`）としてアクセス可能。

## 関数型
`function`
- 関数を型として扱う。
  - 例: `(int, int) => int` は2つの`int`を受け取り、`int`を返す関数。
  - 例: `() => void` は引数を取らず、戻り値もない関数。
- 高階関数やラムダ式をサポート。

## オブザーバブル型
`observable<T>`
- 値の変更を監視できる型。
  - メソッド:
    - `subscribe(callback: (T) => void): void`: 値の変更時に呼び出されるコールバックを登録。
    - `unsubscribe(callback: (T) => void): void`: 登録されたコールバックを解除。
  - 例:
    ```
    const obs: observable<int> = observable(0);
    obs.subscribe((value) => console.log(value));
    obs.next(1); // コンソール出力: 1
    ```

## サブジェクト型
`subject<T>`
- `observable`を拡張し、値を発行できる型。
  - メソッド:
    - `next(value: T): void`: 値を更新し、登録されたコールバックを呼び出す。
    - `subscribe(callback: (T) => void): void`: 値の変更時に呼び出されるコールバックを登録。
  - 例:
    ```
    const subj: subject<int> = subject(0);
    subj.subscribe((value) => console.log(value));
    subj.next(1); // コンソール出力: 1
    ```

## リアクティブプロパティ型
`ReactiveProperty<T>`
- プロパティの変更を監視し、リアクティブな動作を提供する型。
  - メソッド:
    - `get(): T`: 現在の値を取得。
    - `set(value: T): void`: 値を設定し、変更を通知。
    - `subscribe(callback: (T) => void): void`: 値の変更時に呼び出されるコールバックを登録。
  - 例:
    ```
    const prop: ReactiveProperty<string> = ReactiveProperty('John');
    prop.subscribe((value) => console.log('Name changed to:', value));
    prop.set('Jane'); // コンソール出力: Name changed to: Jane
    ```

## Null型
`null`
- 値が存在しないことを示す型。
  - 例:
    ```
    const value: null = null;
    ```

## 未定義型
`undefined`
- 値が未定義であることを示す型。
  - 例:
    ```
    let value: undefined;
    console.log(value); // undefined
    ```

## 標準メソッド
### 文字列型 (`string`)
- `length`: 文字列の長さを取得。
- `toUpperCase()`: 文字列を大文字に変換。
- `toLowerCase()`: 文字列を小文字に変換。
- `includes(substring: string): bool`: 部分文字列が含まれているかを判定。
- `replace(search: string, replacement: string): string`: 指定した文字列を置換。
- `split(separator: string): string[]`: 指定した区切り文字で文字列を分割。

### 配列型 (`<TYPE_NAME>[]`)
- `length`: 配列の長さを取得。
- `push(item: <TYPE_NAME>): void`: 配列の末尾に要素を追加。
- `pop(): <TYPE_NAME>`: 配列の末尾から要素を削除して返す。
- `map(callback: (<TYPE_NAME>) => <NEW_TYPE>): <NEW_TYPE>[]`: 各要素に関数を適用して新しい配列を生成。
- `filter(callback: (<TYPE_NAME>) => bool): <TYPE_NAME>[]`: 条件を満たす要素のみを含む新しい配列を生成。
- `reduce(callback: (<ACCUMULATOR>, <TYPE_NAME>) => <ACCUMULATOR>, initialValue: <ACCUMULATOR>): <ACCUMULATOR>`: 配列を1つの値に集約。

### オブジェクト型 (`Record<<TYPE_A>, <TYPE_B>>`)
- `keys(obj: Record<<TYPE_A>, <TYPE_B>>): <TYPE_A>[]`: オブジェクトのキーを配列として取得。
- `values(obj: Record<<TYPE_A>, <TYPE_B>>): <TYPE_B>[]`: オブジェクトの値を配列として取得。
- `entries(obj: Record<<TYPE_A>, <TYPE_B>>): [<TYPE_A>, <TYPE_B>][]`: キーと値のペアを配列として取得。
- `hasOwnProperty(key: <TYPE_A>): bool`: 指定したキーがオブジェクトに存在するかを判定。
